# Evidence Directory

## Approach

This pack uses **deterministic engine outputs** as primary evidence rather than portal screenshots.

**Script outputs** prove logic, methodology, and reproducibility.
**Portal screenshots** prove environment access and will be added when running against live Azure infrastructure.

## File Types

| Extension | Type | Purpose |
|-----------|------|---------|
| `.txt` | Engine output | Primary evidence — real script, real output |
| `.html` | Reference dashboard | Visual reference of expected portal state |
| `.md` | Report output | Formatted engine report |
| `.png` | Portal screenshot | Added when running against live environment |
